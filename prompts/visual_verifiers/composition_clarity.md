# Composition Clarity

Screenshots are bundled for multiple viewport widths (typically 360 px mobile and 820 px prose column). Evaluate the composition at **each** supplied width — readers see both.

Pass only if the layout has a clear hierarchy, readable spacing, and an obvious focal point at every supplied width. The narrow view must be a genuine re-composition (stacked cards, simplified chart, reflowed columns) — not a shrunken copy of the wide view with microscopic type, squashed columns, cramped padding, or a horizontally-scrolling wide table. Fail if the composition feels cluttered, unbalanced, hard to scan, or visually noisy at any width — or if the narrow view is clearly just the desktop design scaled down rather than redesigned for the width.
